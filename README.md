# Functional LinkedIn Clone 

## Stack used
- **React.js**
- **Redux**
- **Firebase Authentication**
- **Firebase Firestore**

## Usage:
- **Sign-up** 
  - Enter your email, full name and password. 
- **Log-in**: 
  - Enter your email and password.
